Chào mừng đến với trang web của tôi

Xin chào! Đây là trang web của tôi. Trang web này được thiết kế đẹp và sử dụng hình nền sinh động để tạo nên không khí ấn tượng. Hãy khám phá trang web của tôi và tìm hiểu thêm về những gì tôi có để chia sẻ.

Khám phá ngay
#include #include int main() { std::string name; std::cout << "Xin chào! Xin vui lòng nhập tên của bạn: "; std::getline(std::cin, name); std::cout << "Chào mừng " << name << " đến với chương trình C++ đầu tiên của bạn!" << std::endl; return 0; }
